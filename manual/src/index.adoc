= Create: Factory Controller Manual

Create: Factory Controller introduces a new block, the "Factory Controller". It allows you to place factory gauges on a virtual board stored in a single block, instead of having a wall of gauges in the game world. The factory gauges are managed via a GUI. The mod adds many enhancements to factory gauges, and some other components may also be placed in factory controllers.
